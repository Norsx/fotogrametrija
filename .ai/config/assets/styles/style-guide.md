# Style guide

## Writing

- Prefer concise, evidence-backed statements.
- Use consistent terminology across chapters.

## Figures and tables

- Number sequentially (`Figure 1`, `Table 1`).
- Every figure/table must have a caption and in-text reference.

## Visualization defaults

- Primary color: `#2E5AAC`
- Accent color: `#D67A1F`
- Neutral text: `#1F2937`

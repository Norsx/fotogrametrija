# Gemini CLI Instructions (LiteRealm Core Rules)

1. **PROAKTIVNI GIT (CRITICAL):** Nakon SVAKE logičke cjeline, obavezno napravi `git add` i `git commit` (s prefiksom `🤖 [AI]`). Ne čekaj korisnika.
2. **Read-Only Podaci:** Nikada ne mijenjaj podatke u `data/raw/`. Smiješ ih samo čitati.
3. **Kontekst:** Prije rada, OBAVEZNO pročitaj `STATE.md` (za trenutni zadatak) i `.ai/config/project.yaml` (za LaTeX metapodatke).
4. **Detalji:** `.ai/config/AGENTS.md` je referenca. Čitaj taj dokument SAMO ako ti trebaju tehnički detalji o RAG-u, build skriptama ili definicijama agenata. Ne čitaj ga preventivno za opća pravila.

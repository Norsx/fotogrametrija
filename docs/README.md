# docs/

**Your project documentation lives here.** This directory is exclusively reserved for:

- Project reports, seminar papers, and theses
- LaTeX document source files and configurations
- Diagrams, schemas, and technical notes

---

## 🎓 Academic Templates
All standard FSB academic templates (Seminar, Thesis, Paper) and their manuals have been moved to:
👉 **`.ai/templates/`**

This keeps your root `/docs` directory pristine and ready for your own project-specific writing.

> 💡 **Tip**: To build your LaTeX documents automatically into PDFs, use the built-in compiler script:
> - **Windows (PowerShell)**: `.\.ai\scripts\helpers\build-docs.ps1`
> - **Linux/macOS (Bash)**: `./.ai/scripts/helpers/build-docs.sh`
